This repo is a forked repo from mobz for a plugin that allows ElasticSearch be presented in the NodeJS environment

Note: the original plugin installation requires NodeJS environment be set up first, the 3 steps instruction can refer to:

https://tecadmin.net/install-latest-nodejs-and-npm-on-centos/ 

1. yum install -y gcc-c++ make
2. curl -sL https://rpm.nodesource.com/setup_10.x | sudo -E bash -
3. sudo yum install nodejs
